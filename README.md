# AppointmentSystemApplication
نظام حجز المواعيد (Appointment Booking System)  نظام Back-End مبني باستخدام Spring Boot لإدارة وحجز المواعيد. يدعم تسجيل المستخدمين باستخدام JWT، التحقق من توفر الوقت، تحديث حالة المواعيد، وإرسال إشعارات فورية عبر WebSocket. المشروع مبني بمعمارية طبقية ويستخدم Spring Data JPA و MySQL
